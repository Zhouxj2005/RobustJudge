## 1. 计算相对SEM的时候，“先计算SEM再归一化”和“先归一化再求SEM”对于criteria-level和query-level是一样的吗？

> 结论：完全一样

#### 1.1. Criteria-level 的等价性

对指定的 query $i$ 和 rubric $r$：

- **先计算 SEM，再归一化（文档现有做法）**：
  先算打分均值 $\bar{Y}_{ir}^{(n)}$ 的标准误，即 $SEM_{ir}(n)$，然后再除以常数 $P_{ir}$。
  $$
  R^{\text{crit}}_{ir}(n) = \frac{SEM_{ir}(n)}{P_{ir}} 
  $$

- **先归一化，再计算 SEM**：
  先对每次采样的原始分做归一化 $Z_{irt} = \frac{Y_{irt}}{P_{ir}}$，然后再求均值 $\bar{Z}_{ir}^{(n)}$ 的标准误。
  $$
  SEM(\bar{Z}_{ir}^{(n)}) = SEM\left(\frac{\bar{Y}_{ir}^{(n)}}{P_{ir}}\right) = \frac{1}{P_{ir}} SEM(\bar{Y}_{ir}^{(n)}) = \frac{SEM_{ir}(n)}{P_{ir}} 
  $$

**结论：两者完全相等。**

#### 1.2. Query-level 的等价性

对指定的 query $i$：

- **先聚合总分计算 SEM，再归一化（文档现有做法）**：
  先算总分均值 $\bar{S}_i^{(n)}$ 的标准误，即 $SEM_i^{\text{query}}(n)$，然后再除以常数 $P_i$。
  $$
  R_i^{\text{query}}(n) = \frac{SEM_i^{\text{query}}(n)}{P_i} 
  $$

- **先聚合总分并除以总满分归一化，再计算 SEM**：
  每次采样先算归一化总分 $W_{it} = \frac{S_{it}}{P_i}$，然后算均值 $\bar{W}_i^{(n)}$ 的标准误。
  $$
  SEM(\bar{W}_i^{(n)}) = SEM\left(\frac{\bar{S}_i^{(n)}}{P_i}\right) = \frac{1}{P_i} SEM(\bar{S}_i^{(n)}) = \frac{SEM_i^{\text{query}}(n)}{P_i} 
  $$

**结论：两者同样完全相等。**

## 2. 修正概率模型，形式化表示多prompt和单prompt采样的均值、方差差异

#### 2.1. 统一的底层概率模型

对于任何一个评估对象（记为 $i$，可以是某个 query 或某个 rubric），大模型在给定 prompt $p \in \mathcal{P}$ 和第 $t$ 次解码下的打分分数 $Y_{ipt}$ 受到三重因素影响。我们将其定义为严格的加性正交模型：

$$ Y_{ipt} = G_i + \beta_i(p) + \varepsilon_{ipt} $$

其中各项的严格数学定义与统计假设如下：

1. **$G_i$ **：对象 $i$ 的内蕴真实得分。定义为**在所有语义等价 prompt 族和无限次解码下的期望**，即 $G_i := \mathbb{E}_{p \sim \mathcal{P}, t}[Y_{ipt}]$。

2. **$\beta_i(p)$**：Prompt 引入的系统漂移，满足：$$ \mathbb{E}_p[\beta_i(p)] = 0, \quad \mathrm{Var}_p(\beta_i(p)) = \sigma_P^2 $$

3. **$\varepsilon_{ipt}$ **：给定 prompt 后的解码噪声，满足：$$ \mathbb{E}_t[\varepsilon_{ipt} \mid p] = 0, \quad \mathrm{Var}_t(\varepsilon_{ipt} \mid p) = \sigma_E^2 $$

   >  **正交性**：抽取的 prompt $p$ 与模型产生的解码噪声 $\varepsilon$ **相互独立（？不独立应该）**。

#### 2.2  现象 A：多 Prompt 一致性的 Bias-Variance Crossover

$$ \mathrm{MSE}(n) = \mathbb{E}\left[ (\bar{Y}^{(n)} - G_i)^2 \right] = \mathrm{Bias}^2 + \mathrm{Variance} $$

单 prompt 的 MSE 为：
$$
\mathrm{MSE}_s(n) = \beta_i(p_0)^2 + \frac{\sigma_E^2(p_0)}{n}
$$

多 prompt 的均方误差为（**$p$是有限采样，不能套E，不能用0**）：
$$
\mathrm{MSE}_m(n) = 0^2 + \frac{\sigma_P^2 + E_p(\sigma_E^2(p))}{n}
$$
解得严格的相变阈值：
$$
n^\star = \frac{\sigma_P^2}{\beta_i(p_0)^2}
$$

#### 2.3. 现象 B：Query-level 相对 SEM 小于 Criteria-level 的空间聚合效应

引入两个直接基于实验定义的符号：

1. **打分误差相关系数 $\rho_{rs}$**：
   $$
   \rho_{rs} = \frac{\mathrm{Cov}(\bar{Y}_{ir}^{(n)}, \bar{Y}_{is}^{(n)})}{SEM_{ir}(n) SEM_{is}(n)} \in [-1, 1]
   $$
   
2. **分数占比权重 $w_{ir}$**：
   $$
   w_{ir} = \frac{P_{ir}}{P_i}，其中P_i = \sum_{r=1}^{m_i} P_{ir} ，满足 \sum_r w_{ir} = 1
   $$
   

$query-level$ 相对 $SEM$
$$
R_i^{\text{query}}(n) = \sqrt{ \sum_{r=1}^{m_i} \left( w_{ir} R^{\text{crit}}_{ir}(n) \right)^2 + \sum_{r\neq s} \rho_{rs} \left( w_{ir} R^{\text{crit}}_{ir}(n) \right) \left( w_{is} R^{\text{crit}}_{is}(n) \right) }
$$
将根号内的 $\rho_{rs}$ 全部放大替换为 $1$，即可得到一个严格的上界：

$$
R_i^{\text{query}}(n) < \sqrt{ \sum_{r=1}^{m_i} \left( w_{ir} R^{\text{crit}}_{ir}(n) \right)^2 + \sum_{r\neq s} \left( w_{ir} R^{\text{crit}}_{ir}(n) \right) \left( w_{is} R^{\text{crit}}_{is}(n) \right) } 
$$


不等式右侧恰好是多项式完全平方的展开式 $\left( \sum_{r} x_r \right)^2$。我们将其收拢并开方：

$$
R_i^{\text{query}}(n) < \sqrt{ \left( \sum_{r=1}^{m_i} w_{ir} R^{\text{crit}}_{ir}(n) \right)^2 } = \sum_{r=1}^{m_i} w_{ir} R^{\text{crit}}_{ir}(n)
$$

+ **结论：**一个 Query 总分的相对 SEM，**永远小于或等于**其内部各个 Rubric 相对 SEM 的**加权平均值**

考察**加权平均与算术平均的关系**
$$
\sum_{r=1}^{m_i} w_{ir} R^{\text{crit}}_{ir}(n) - \frac{1}{m_i} \sum_{r=1}^{m_i} R^{\text{crit}}_{ir}(n) = (m_i - 1) \cdot \text{Cov}\left(w_{ir}, R^{\text{crit}}_{ir}(n)\right)
$$
因此，只要满足以下**两个条件之一**，结论即可严格成立：

1. **等权条件**：若同一 query 内的所有 **rubric 满分相同**（即 $P_{ir}$ 皆相等），则**加权平均等于算术平均**，严格不等式必然成立。
2. **误差非正相关条件**：在真实场景中，一个 rubric 的分数权重 $w_{ir}$ 通常与其相对测量误差 $R^{\text{crit}}_{ir}$ 是**不相关的**（即 $\text{Cov} \approx 0$），或者**满分越高的 rubric 相对误差反而越小**（即 $\text{Cov} < 0$）。在这两种情况下，加权平均 $\le$ 算术平均。

## 4. 增设实验实际考察的影响因素和结果优化

![image-20260410202441137](C:\Users\Zhouxj's Lenovo\AppData\Roaming\Typora\typora-user-images\image-20260410202441137.png)

**feature、targer都是query级（或rubric级聚合成query级）:一个query一个值，形成3*100维的向量**

+ `rubric_list_count`：当前 query 的 rubric 条目数量。
+ `total_points`：当前 query 的总分上限，也就是**所有 rubric 的 points 之和**。
+ `prompt_words`：原始 question 的词数。
+ `response_words`：生成回答的词数。
+ `gt_norm_score`：Ground Truth 的归一化总分，表示 GT 总分占该 query 总满分的比例。
+ `gt_midness`：Ground Truth 离“中间分”有多近。越接1近中间值 0.5，这个值越大；越接近 0 或 1，这个值越小。

$$
gt\_midness=1-\frac{abs(norm_{score}-0.5)}{0.5}
$$

3. query-level 的总分波动整体比 item-level 更难被这些 feature 解释，说明总分聚合确实在一定程度上抵消了局部 rubric 的随机性。
