# 统一双重随机效应模型下的鲁棒性数学推导

为了在数学上严谨地解释“多 prompt 的一致性反超”与“query-level 相对 SEM 的衰减”这两个预实验现象，我们抛弃了对真实能力（Ground Truth）的启发式外部假设，转而采用心理测量学（Psychometrics）概化理论中的**方差分量模型（Variance Component Model）**，建立一个底层的生成式概率模型。

## 1. 统一的底层概率模型

对于任何一个评估对象（记为 $i$，可以是某个 query 或某个 rubric），大模型在给定 prompt $p \in \mathcal{P}$ 和第 $t$ 次解码下的打分分数 $Y_{ipt}$ 受到三重因素影响。我们将其定义为严格的加性正交模型：

$$ Y_{ipt} = G_i + \beta_i(p) + \varepsilon_{ipt} $$

其中各项的严格数学定义与统计假设如下：

1. **$G_i$ (Intrinsic Ground Truth)**：对象 $i$ 的内蕴真实得分。定义为**在所有语义等价 prompt 族和无限次解码下的期望**，即 $G_i := \mathbb{E}_{p \sim \mathcal{P}, t}[Y_{ipt}]$。
2. **$\beta_i(p)$ (Prompt-level Effect)**：Prompt 引入的系统漂移。它是一个仅依赖于 prompt $p$ 的随机变量，满足：
   $$ \mathbb{E}_p[\beta_i(p)] = 0, \quad \mathrm{Var}_p(\beta_i(p)) = \sigma_P^2 $$
   
   >  $\sigma_P^2$ 代表了该任务对 prompt 的敏感度。
3. **$\varepsilon_{ipt}$ (Decoding Noise)**：给定 prompt 后的解码噪声。它满足：
   $$ \mathbb{E}_t[\varepsilon_{ipt} \mid p] = 0, \quad \mathrm{Var}_t(\varepsilon_{ipt} \mid p) = \sigma_E^2 $$
   
   > $\sigma_E^2$ 代表了模型温度带来的解码不确定性。 
4. **正交性**：抽取的 prompt $p$ 与模型产生的解码噪声 $\varepsilon$ 相互独立。

基于这个极简且自洽的底层模型，我们分别推导现象 A（多 Prompt 交叉阈值）和现象 B（Query-level 聚合衰减）。

---

## 2. 现象 A：多 Prompt 一致性的 Bias-Variance Crossover

我们比较两种采样策略在重复 $n$ 次并求均值后的**均方误差（Mean Squared Error, MSE）**。
$$ \mathrm{MSE}(n) = \mathbb{E}\left[ (\bar{Y}^{(n)} - G_i)^2 \right] = \mathrm{Bias}^2 + \mathrm{Variance} $$

> $Bias=\mathbb{E}_t\left[ \bar{Y}^{(n)} - G_i \right],\mathrm{Variance}=\mathrm{Var}_t\left[ \bar{Y}^{(n)}\right]$
>
> 第二步推导是一个公式，为了精简未证明

### 2.1 单 Prompt 策略 (Single-Prompt: Fixed Effect)

人类开发者**随机抽取了一个** prompt $p_0$ 并将其固定。后续的 $n$ 次采样仅改变解码过程 $t$。
其打分均值为：
$$
\bar{Y}_{s}^{(n)} = \frac{1}{n}\sum_{t=1}^n Y_{i p_0 t} = G_i + \beta_i(p_0) + \frac{1}{n}\sum_{t=1}^n \varepsilon_{i p_0 t}
$$

- **偏差 (Bias)**：$\mathbb{E}_t[\bar{Y}_{s}^{(n)}] - G_i = \beta_i(p_0)$。注意，$\beta_i(p_0)$ 对于这个评价过程是一个固化的非零常数，**不会随 $n$ 增加而衰减**。
- **方差 (Variance)**：由于固定的常数 $\beta_i(p_0)$ 不产生方差，打分均值的方差仅来自独立的解码噪声：$\mathrm{Var}_t(\bar{Y}_{s}^{(n)}) = \frac{\sigma_E^2}{n}$。

因此，单 prompt 的 MSE 为：
$$
\mathrm{MSE}_s(n) = \beta_i(p_0)^2 + \frac{\sigma_E^2}{n}
$$


### 2.2 多 Prompt 策略 (Multi-Prompt: Random Effect)

每次采样时，均从 $\mathcal{P}$ 中重新独立抽取 prompt $p_k$，并进行独立的解码 $t_k$。
其 $n$ 次打分均值为：
$$
\bar{Y}_{m}^{(n)} = \frac{1}{n}\sum_{k=1}^n Y_{i p_k t_k} = \frac{1}{n}\sum_{k=1}^n \Big( G_i + \beta_i(p_k) + \varepsilon_{i p_k t_k} \Big)
$$

- **偏差 (Bias)**：
  因为期望具有线性可加性，且 $\mathbb{E}_p[\beta_i(p)] = 0$：
  $$
  \mathbb{E}_{p,t}[\bar{Y}_{m}^{(n)}] = G_i + \frac{1}{n}\sum_{k=1}^n \underbrace{\mathbb{E}_p[\beta_i(p_k)]}_{=0} + \frac{1}{n}\sum_{k=1}^n \underbrace{\mathbb{E}_t[\varepsilon_{i p_k t_k}]}_{=0} = G_i
  $$
  因此偏差为 $0$。**系统偏差通过大样本平均被彻底消除**。
  
- **方差 (Variance) 推导**：
  在多 prompt 策略中，第 $k$ 次采样的 $(p_k, t_k)$ 与第 $j$ 次采样的 $(p_j, t_j)$ 是独立同分布的（I.I.D.）。同时，单次采样内部的 prompt 与解码噪声正交（独立）。因此：
  $$
  \mathrm{Var}_{p,t}(\bar{Y}_{m}^{(n)}) = \frac{1}{n^2} \sum_{k=1}^n \mathrm{Var}\Big( \beta_i(p_k) + \varepsilon_{i p_k t_k} \Big) 
   = \frac{1}{n^2} \sum_{k=1}^n \Big( \mathrm{Var}_p(\beta_i(p_k)) + \mathrm{Var}_t(\varepsilon_{i p_k t_k}) \Big)
  $$
  
  代入我们在模型中定义的方差：
  $$
  \mathrm{Var}_{p,t}(\bar{Y}_{m}^{(n)}) = \frac{1}{n^2} \sum_{k=1}^n (\sigma_P^2 + \sigma_E^2) = \frac{\sigma_P^2 + \sigma_E^2}{n} 
  $$
  

因此，多 prompt 的均方误差为：
$$
\mathrm{MSE}_m(n) = 0^2 + \frac{\sigma_P^2 + \sigma_E^2}{n}
$$


### 2.3 严谨推导相变阈值 (The Crossover Theorem)

多 Prompt 何时优于单 Prompt？只需令 $\mathrm{MSE}_m(n) < \mathrm{MSE}_s(n)$：
$$
\frac{\sigma_P^2 + \sigma_E^2}{n} < \beta_i(p_0)^2 + \frac{\sigma_E^2}{n}
$$
两边消去相同的解码噪声项 $\frac{\sigma_E^2}{n}$，我们得到一个极其简洁的不等式：
$$
\frac{\sigma_P^2}{n} < \beta_i(p_0)^2
$$


解得严格的相变阈值：
$$
n^\star = \frac{\sigma_P^2}{\beta_i(p_0)^2}
$$


**推论的物理意义**：
从底层推导出多 prompt 的单次方差必然更大（多了 $\sigma_P^2$）。当 $n < n^\star$ 时，方差主导，单 prompt 占优；当 $n > n^\star$ 时，偏差主导，多 prompt 通过消解固化偏差实现一致性反超。这就完美解释了实验中多 prompt 的 Spearman 表现“先低后高”。

## 3. 现象 B：Query-level 相对 SEM 小于 Criteria-level 的空间聚合效应

在统计学中，标准误（SEM）的数学定义即为“采样均值的标准差”。因此，不论采用何种采样策略，第 $i$ 个 query 下第 $r$ 个 rubric 在 $n$ 次采样并求平均后的标准误都可严谨地写为：
$$ SEM_{ir}(n) = \sqrt{\mathrm{Var}(\bar{Y}_{ir}^{(n)})} $$

### 3.1 Criteria-level 相对 SEM 的表达式

根据实验定义，第 $r$ 个 rubric 的满分为 $P_{ir}$，其 criteria-level 相对 SEM 为：
$$
R^{\text{crit}}_{ir}(n) = \frac{SEM_{ir}(n)}{P_{ir}}
$$
将其变形，绝对标准误可表示为满分与其相对误差的乘积：
$$
SEM_{ir}(n) = P_{ir} \cdot R^{\text{crit}}_{ir}(n)
$$


### 3.2 Query-level 聚合方差与权重定义

对于第 $i$ 个 query，其单次打分总分为 $S_{it} = \sum_{r=1}^{m_i} Y_{irpt}$，经过 $n$ 次采样后的均值 $\bar{S}_i^{(n)} = \sum_{r=1}^{m_i} \bar{Y}_{ir}^{(n)}$。
根据多变量方差的展开公式，query 均值总分的方差为：
$$
\mathrm{Var}(\bar{S}_i^{(n)}) = \sum_{r=1}^{m_i} \mathrm{Var}(\bar{Y}_{ir}^{(n)}) + \sum_{r\neq s} \mathrm{Cov}(\bar{Y}_{ir}^{(n)}, \bar{Y}_{is}^{(n)})
$$


引入两个直接基于实验定义的符号：
1. **打分误差相关系数 $\rho_{rs}$**：$\rho_{rs} = \frac{\mathrm{Cov}(\bar{Y}_{ir}^{(n)}, \bar{Y}_{is}^{(n)})}{SEM_{ir}(n) SEM_{is}(n)} \in [-1, 1]$
2. **分数占比权重 $w_{ir}$**：$w_{ir} = \frac{P_{ir}}{P_i}$，其中 $P_i = \sum_{r=1}^{m_i} P_{ir}$ 是 Query 总满分，满足 $\sum_r w_{ir} = 1$。

代入相关系数，Query 总分的绝对标准误（SEM）满足：
$$
\left( SEM_i^{\text{query}}(n) \right)^2 = \mathrm{Var}(\bar{S}_i^{(n)}) =\sum_{r=1}^{m_i} SEM_{ir}^2(n) + \sum_{r\neq s} \rho_{rs} SEM_{ir}(n) SEM_{is}(n)
$$


### 3.3 相对 SEM 的空间聚合推导

为了计算 Query-level 相对 SEM $R_i^{\text{query}}(n) = \frac{SEM_i^{\text{query}}(n)}{P_i}$，我们将 3.1 节得到的代换式 $SEM_{ir}(n) = P_{ir} R^{\text{crit}}_{ir}(n) = P_i w_{ir} R^{\text{crit}}_{ir}(n)$ 代入上式根号内：

$$
SEM_i^{\text{query}}(n) = \sqrt{ \sum_{r=1}^{m_i} \left( P_i w_{ir} R^{\text{crit}}_{ir}(n) \right)^2 + \sum_{r\neq s} \rho_{rs} \left( P_i w_{ir} R^{\text{crit}}_{ir}(n) \right) \left( P_i w_{is} R^{\text{crit}}_{is}(n) \right) }
$$


将公因式 $P_i$ 从根号中完美提取并移至左侧作除法，直接得到：
$$
R_i^{\text{query}}(n) = \sqrt{ \sum_{r=1}^{m_i} \left( w_{ir} R^{\text{crit}}_{ir}(n) \right)^2 + \sum_{r\neq s} \rho_{rs} \left( w_{ir} R^{\text{crit}}_{ir}(n) \right) \left( w_{is} R^{\text{crit}}_{is}(n) \right) }
$$


### 3.4 核心不等式放缩与抵消效应

在 LLM 评估中，模型对不同 rubric 指标（如逻辑性、流畅度等）的评价噪声不可能达到极端的完全正相关，即 $\rho_{rs} < 1$ 恒成立。将根号内的 $\rho_{rs}$ 全部放大替换为 $1$，即可得到一个严格的上界：

$$
R_i^{\text{query}}(n) < \sqrt{ \sum_{r=1}^{m_i} \left( w_{ir} R^{\text{crit}}_{ir}(n) \right)^2 + \sum_{r\neq s} \left( w_{ir} R^{\text{crit}}_{ir}(n) \right) \left( w_{is} R^{\text{crit}}_{is}(n) \right) } 
$$


不等式右侧恰好是多项式完全平方的展开式 $\left( \sum_{r} x_r \right)^2$。我们将其收拢并开方：

$$
R_i^{\text{query}}(n) < \sqrt{ \left( \sum_{r=1}^{m_i} w_{ir} R^{\text{crit}}_{ir}(n) \right)^2 } = \sum_{r=1}^{m_i} w_{ir} R^{\text{crit}}_{ir}(n)
$$


**推论的物理意义**：
推导在数学上严格证明了，Query 总分的相对 SEM 必然严格小于其内部各个 Rubric 相对 SEM 的**加权平均值**。只要不同 rubric 之间存在不完全相关的随机误差（$\rho < 1$），这种聚合就会引发“**方差抵消效应**”。在合理假设下（满分权重与相对误差不严重正相关），全局求平均后必然会观察到实验现象：$\overline{R}^{\text{query}}(n) < \overline{R}^{\text{crit}}(n)$。

---

### 总结

基于底层的双重随机效应（加性正交）模型，我们从数学上证明：
1. **现象 A (多 Prompt)** 利用了**时间维度**的随机效应累加，通过大样本清除了单 Prompt 固化的系统偏差（$\mathbb{E}[\beta]=0$），从而在阈值 $n^\star$ 之后实现一致性反超。
2. **现象 B (Query 聚合)** 利用了**空间维度**的协方差耗散，通过不同维度打分噪声的非完全正相关（$\rho < 1$）产生方差抵消，从底层机制上保证了聚合评估的相对误差衰减。
